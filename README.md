# 👋 프론트엔드 개발자 전상원입니다

사용자의 행동 흐름을 이해하고, 더 나은 경험으로 개선하는 UI를 만드는 것을 목표로 합니다.
단순한 기능 구현이 아닌 **문제 해결 중심의 개발**을 지향합니다.

## 🔍 Summary

* React + TypeScript 기반 웹 서비스 개발 경험
* Firebase 기반 서버리스 아키텍처 설계 및 데이터 처리 경험
* 사용자 경험 개선을 위한 상태 관리 및 비동기 처리 최적화
* 재사용성과 유지보수성을 고려한 컴포넌트 설계

## 🚀 Projects

### 1. Find Player Game

> 실시간 매칭 기반 게임 커뮤니티 플랫폼

* Firebase Realtime Database + Cloud Functions를 활용한 서버리스 구조 설계
* GitHub Actions를 활용한 **데이터 동기화 자동화 환경 구축**
* TypeScript 기반 API 데이터 명시로 **타입 안정성 및 개발 생산성 향상**

👉 **성과**

* 실시간 상태 기반 UI 업데이트 구조 설계 경험 확보
* 비동기 데이터 흐름 관리 및 예외 처리 로직 개선

[🔗 GitHub Repository](여기에_링크)

### 2. Xflix

> OTT 서비스 UI/UX 클론 및 기능 확장 프로젝트
